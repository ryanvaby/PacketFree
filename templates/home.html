This is PacketFree, our easy to use pseudo packet sniffer. By sending several "SYN-ACK" packets, which are essentially what a computer uses to connect to another server, we are able to discern the round trip time of a packet to that server. This information is useful for determing network congestion, and determining how well your web server can handle network traffic! To begin, please enter either "My own," if you'd like to test your own web domains, or "Yours," if you'd like to use our pre-loaded domains!
